# Project Design Principles: Frontend Design

This project adheres to high-quality frontend design standards to avoid generic aesthetics and deliver a polished, production-grade experience.

## Design Thinking
- **Purpose**: Stripe Connect Multi-vendor Marketplace.
- **Tone**: Editorial & Refined. High-contrast typography, generous whitespace, and a minimalist but intentional layout.
- **Differentiation**: Uses a "Museum/Archive" aesthetic for a marketplace, treating products as curated entries rather than just commodity listings.

## Guidelines
- **Typography**: Uses 'Libre Baskerville' for an editorial serif feel, paired with 'Inter' and 'JetBrains Mono' for technical precision.
- **Color & Theme**: A sophisticated palette of Slate, Off-white (#F9F8F4), and Deep Charcoal.
- **Motion**: Subtle staggered reveals and transition effects using `motion/react`.
- **Spatial Composition**: Asymmetrical layouts, editorial grids, and intentional negative space.
- **Backgrounds & Visual Details**: Grain-like textures (via color choices), delicate borders, and high-impact imagery.

*The following is the full specification for the Frontend Design philosophy adopted by this project:*

### name: frontend-design
### description: Create distinctive, production-grade frontend interfaces with high design quality.

This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

**Typography**: Choose fonts that are beautiful, unique, and interesting. Avoid generic fonts like Arial and Inter; opt instead for distinctive choices that elevate the frontend's aesthetics; unexpected, characterful font choices. Pair a distinctive display font with a refined body font.

**Color & Theme**: Commit to a cohesive aesthetic. Use CSS variables for consistency. Dominant colors with sharp accents outperform timid, evenly-distributed palettes.

**Motion**: Use animations for effects and micro-interactions. Prioritize CSS-only solutions for HTML. Use Motion library for React when available. Focus on high-impact moments.

**Spatial Composition**: Unexpected layouts. Asymmetry. Overlap. Diagonal flow. Grid-breaking elements. Generous negative space OR controlled density.
